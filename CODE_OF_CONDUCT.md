# Code of Conduct

This is a small hobby project. The expectation is simple: be respectful and
constructive in issues, pull requests, and discussions. No harassment,
personal attacks, or discriminatory language.

Disagreements about design or code are fine and expected - keep them about
the work, not the person.

If someone is being disruptive or abusive, maintainers may edit, lock, or
remove comments/issues/PRs, and may block repeat offenders from the repo.

To report a problem, open an issue or contact the maintainer directly through
GitHub.
